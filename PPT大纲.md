## 第8页：技术难点 - JWT认证

**标题**：技术难点一：身份认证与权限管理

**问题描述**：
```
挑战：
└─ 如何区分普通用户和管理员？
└─ 如何保证Token安全性？
└─ 如何处理Token过期？
```

**解决方案**：
```javascript
// 1. Token生成（包含用户ID和角色）
const token = jwt.sign(
  { id: user.id, role: user.role },
  process.env.JWT_SECRET,
  { expiresIn: '7d' }  // 7天有效期
);

// 2. Token验证中间件
const verifyAdminToken = (req, res, next) => {
  const token = req.headers.authorization?.replace('Bearer ', '');
  const decoded = jwt.verify(token, process.env.JWT_SECRET);
  if (decoded.role !== 'admin') {
    return res.json({ message: '需要管理员权限' });
  }
  next();
};

// 3. 前端自动添加Token到请求头
axios.interceptors.request.use(config => {
  config.headers.Authorization = `Bearer ${token}`;
  return config;
});
```

**技术亮点**：
- ✅ Token自动过期（7天）
- ✅ 自动检测Token无效并跳转登录
- ✅ 角色权限控制（admin/user）

## 第9页：技术难点 - 图片处理

**标题**：技术难点二：智能图片处理系统

**问题描述**：
```
挑战：
└─ 用户输入多种图片URL格式
└─ 数据库存储JSON字符串
└─ 避免双重JSON编码问题
```

**支持的格式**：
```
格式1：单个URL
输入：http://example.com/1.jpg
存储：["http://example.com/1.jpg"]

格式2：多个URL（逗号分隔）
输入：url1.jpg, url2.jpg, url3.jpg
存储：["url1.jpg", "url2.jpg", "url3.jpg"]

格式3：JSON数组字符串
输入：["url1.jpg", "url2.jpg"]
存储：["url1.jpg", "url2.jpg"]
```

**解决方案**：
```javascript
// 智能解析器
let processedImages;
if (typeof images === 'string') {
  try {
    processedImages = JSON.parse(images);  // 尝试解析JSON
    if (!Array.isArray(processedImages)) {
      processedImages = [images];
    }
  } catch {
    // 检查是否为逗号分隔
    processedImages = images.includes(',')
      ? images.split(',').map(url => url.trim())
      : [images.trim()];
  }
} else if (Array.isArray(images)) {
  processedImages = images;
}
// 统一存储为JSON数组
JSON.stringify(processedImages)
```

**技术亮点**：
- ✅ 支持任意格式输入
- ✅ 自动去除空格
- ✅ 防止双重编码

## 第10页：技术难点 - 文件上传

**标题**：技术难点三：图片上传与存储

**技术方案**：
```
Multer中间件配置：
├─ 存储位置：backend/public/images/
├─ 文件命名：upload-时间戳-随机数.jpg
├─ 文件类型：jpg|jpeg|png|gif|webp
├─ 文件大小：最大5MB
└─ 返回URL：http://localhost:3001/images/xxx.jpg
```

**核心代码**：
```javascript
// Multer配置
const storage = multer.diskStorage({
  destination: 'public/images/',
  filename: (req, file, cb) => {
    const uniqueSuffix = Date.now() + '-' + Math.round(Math.random() * 1E9);
    cb(null, 'upload-' + uniqueSuffix + path.extname(file.originalname));
  }
});

const upload = multer({
  storage: storage,
  limits: { fileSize: 5 * 1024 * 1024 },  // 5MB
  fileFilter: (req, file, cb) => {
    const allowed = /jpeg|jpg|png|gif|webp/;
    allowed.test(file.mimetype) ? cb(null, true) : cb(new Error('只允许图片'));
  }
});

// 路由
app.post('/api/upload/single', upload.single('image'), (req, res) => {
  res.json({ url: `http://localhost:3001/images/${req.file.filename}` });
});
```

**技术亮点**：
- ✅ 自动生成唯一文件名
- ✅ 文件类型和大小限制
- ✅ 支持单图/多图上传
